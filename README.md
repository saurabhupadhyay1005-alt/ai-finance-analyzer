# 💰 AI Personal Finance Analyzer

An intelligent data science project that analyzes financial transaction data and predicts future spending using machine learning.

---

## 🚀 Features

• Upload transaction CSV file
• Analyze spending patterns
• AI-generated spending insights
• Interactive charts and visualizations
• Machine learning spending prediction
• Streamlit dashboard interface

---

## 🧠 Tech Stack

Python
Pandas
NumPy
Scikit-learn
Matplotlib
Streamlit
